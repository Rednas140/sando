# Sando

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- ESLint

## Run Locally

Install dependencies:

```bash
npm install
# or
bun install
```

Start the dev server:

```bash
npm run dev
# or
bun dev
```

Build for production:

```bash
npm run build
# or
bun build
```

Start the production server:

```bash
npm run start
# or
bun start
```

Lint:

```bash
npm run lint
# or
bun lint
```
